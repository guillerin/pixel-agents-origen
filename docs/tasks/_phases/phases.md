# Fases del Proyecto Furniture Shop

## Fase 1: Database & Types (Prioridad Alta)
Migración SQL, seed data, tipos TypeScript compartidos, tipos Go.

## Fase 2: Backend Services (Prioridad Alta)
Servicios Go: CatalogService, PurchaseService, InventoryService, PlacementService.
HTTP handlers, WebSocket handlers, tests unitarios.

## Fase 3: React Shop UI (Prioridad Media)
Componentes React para el webview: catalogo, inventario, compra, placements.

## Fase 4: Angular Admin Panel (Prioridad Media)
CRUD de categorias y productos, dashboard de analytics.

## Fase 5: Integration & Polish (Prioridad Baja)
Integracion VS Code extension, editor de placement, collision detection UI, load testing.
